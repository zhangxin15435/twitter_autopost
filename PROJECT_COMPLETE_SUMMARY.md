## 🔄 系统架构

```
内容管理 (CSV) → 内容读取 → 格式化 → Twitter发布 → 状态更新
     ↓              ↓         ↓          ↓         ↓
  用户编辑        程序读取   智能截取   API发布    自动标记
```

## 📋 使用流程

1. **内容准备** - 用户在CSV文件中添加内容
2. **自动触发** - GitHub Actions定时执行
3. **内容处理** - 系统读取并格式化内容
4. **发布执行** - 自动发布到Twitter
5. **状态更新** - 标记为已发布，避免重复
